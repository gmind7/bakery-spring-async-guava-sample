Spring Async Guava Example
==========================

Spring4.0, SpringBoot0.5, Guava15.0 을 이용한 @Async 예제 입니다.

With eclipse:

```
$ main : SpringBootRunner.java > Run As > Spring Boot App or Java Application
$ test : testSpringBootRunner.java > Run As > JUnit Test 
```

With gradle:

```
$ gradlew bootRun
```
